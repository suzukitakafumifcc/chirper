# Laravel入門

[公式のコース](https://laravel.com/learn/getting-started-with-laravel)
でLaravelを学習します。

## インストール

### 依存関係

```shell
composer install
npm install
```

### 環境変数

```shell
cp .env.example .env
```

### アプリケーションキー

```shell
php artisan key:generate
```

### データベース

```shell
php artisan migrate
```

## 使い方

```shell
composer run dev
```
